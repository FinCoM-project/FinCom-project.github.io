Das Projekt

Im Folgenden geben wir einen groben Überblick über das Projekt!

Ziele

Modellierung des Financial Stress

  1. Identifikation von kurzfristigen Frühwarnindikatoren
  2. Ermittlung von langfristigen strategischen Risikopotenzialen
  3. Ableitung von Handlungsempfehlungen zur Mitigation von finanziellen Risikofaktoren

Datenbasierte Modelle

Bei den datenbasierten Modellen werden die Zusammenhänge und Trends aus den Daten abgeleitet. Die Modelle lernen mit jedem neuen Datensatz dazu und spiegeln so die Realität wider.

Analytische Modelle

Bei den analytischen Modellen handelt es sich um theoriebasierte Prognosemodelle aus der Ökonomie- und Finanzforschung. Diese können durch neue Daten abgeändert oder erweitert werden, grundsätzlich bleibt aber der Theoriebezug erhalten.

Modelle

Methoden

Präskriptive Analytik

Wie können wir die Geschehnisse beeinflussen?
  • Prescriptive Condition Monitoring
  • Analyse der Wirkungsketten
  • Szenario Analyse (Was-wäre-wenn?)
  • Analyse der Vertrauenswürdigkeit der Vorhersage
  • Identifikation von Maßnahmen zur Risikomitigation

Prädiktive Analytik

Was wird passieren?
  • Machine Learning und Regressionsanalysen im finanzwirtschaftlichen Risikomanagement
  • Ableitung von Frühwarnindikatoren und langfristigen Risikopotenzialen

Deskriptive Analytik

Was ist passiert?
  • Analyse der finanzwirtschaftlichen Risikodaten
  • Beschreibung der Variablen (zeitliche Entwicklungen, deskriptive Statistik)
  • Korrelationsanalysen (Identifikation erster Risikofaktoren)
  • Visualisierungstechniken
